# 开始使用
请先[安装](#/doc/install)本组件库。

引入组件

```
import { Switch, Button, Dialog, Tabs } from "y-vue3-ui"
```

注册组件

代码示例：

```
<template>
  <div>
    <Button>按钮</Button>
  </div>
</template>

<script>
import { Switch, Button, Dialog, Tabs } from "y-vue3-ui"
export default {
  components: { Button }
}
</script>
```
